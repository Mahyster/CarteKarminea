Nouvelle version majeure

Ce qui change dans l’Atlas

Version majeure — 3 juillet 2026 V2

L’Atlas devient un véritable outil de navigation RP

L’interface a été retravaillée pour mieux explorer Karminéa, retrouver les lieux utiles, afficher les éléments de carte et guider les joueurs selon leurs besoins.

Explorer

Nouvel Explorer en deux sections

L’Explorer possède désormais deux onglets : Lieux et Éléments de carte. La recherche est commune et permet de retrouver rapidement un commerce, une institution, une rue, un pont, un quartier, une place ou une mer.

Calques

Affichage plus clair des catégories

Le panneau des calques permet d’afficher ou masquer les lieux, mais aussi les routes, ponts, cours d’eau, mers, quartiers et places. Chaque famille peut être gérée séparément.

Carte interactive

Éléments de carte cliquables

Les routes, ponts, quartiers, places, mers et cours d’eau peuvent maintenant avoir leur propre tracé, leur nom au survol et une fiche descriptive dédiée au clic.

Outils

Aller à des coordonnées Minecraft

L’outil de coordonnées permet d’entrer une position X / Z, de centrer la carte dessus, d’afficher une cible et de copier un lien de partage. La cible disparaît proprement à la fermeture de l’outil.

Nouveau

Parcours utiles : trouver un lieu selon son besoin

Depuis le menu Outils, il est maintenant possible de chercher un lieu sans connaître son nom : Acheter, faire des démarches, se soigner ou se divertir. Les résultats ouvrent directement la fiche du lieu et centrent la carte.

Remarques

Un panneau de retour plus simple

L’onglet Remarque permet de générer rapidement un message Discord pour signaler un bug, proposer une idée, corriger une description ou demander l’ajout d’un lieu.

Création

Mode création réservé à l’édition

Le mode création est désormais masqué sur le lien public classique. Il apparaît uniquement avec le paramètre dédié et permet de générer proprement le JSON des lieux et des éléments de carte.

Navigation

Sélecteur de cartes

Un volet déroulant a été ajouté dans l’en-tête pour préparer les futures cartes de Médusa et Kopia. Pour le moment, seule la carte de Karminéa est active.

Optimisations

Interface nettoyée et aide mise à jour

L’ancien mode placement a été retiré, les panneaux se ferment plus proprement entre eux, l’aide a été actualisée et l’accueil présente mieux le rôle de l’Atlas.

Aide rapide

Comment utiliser l’Atlas ?

🖱️

Déplacer la carte

Maintiens le clic gauche sur la carte puis déplace ta souris pour naviguer librement dans Karminéa.

＋

Zoomer et recentrer

Utilise la molette de la souris ou les boutons −, + et Recentrer dans la barre du haut pour ajuster la vue.

☰

Explorer les lieux

Ouvre l’onglet Explorer à gauche pour rechercher un commerce, une institution, un lieu de divertissement ou un service de santé.

⇄

Passer des lieux aux éléments de carte

Dans l’Explorer, utilise les onglets Lieux et Éléments de carte pour passer rapidement des marqueurs aux rues, ponts, quartiers, mers, cours d’eau et places.

🔎

Rechercher et filtrer

La barre de recherche permet de retrouver rapidement un lieu ou un élément de carte. Les filtres permettent d’afficher seulement les catégories utiles.

◎

Afficher ou masquer les calques

Le panneau en haut à droite permet d’afficher ou masquer les lieux, les routes, les ponts, les cours d’eau, les mers, les quartiers et les places.

✦

Consulter une fiche

Clique sur un marqueur ou sur un élément de carte pour ouvrir sa fiche descriptive : nom, catégorie, quartier, description et coordonnées quand elles sont disponibles.

⌖

Aller à des coordonnées

Ouvre le menu Outils, puis l’outil Aller à des coordonnées pour entrer une position X / Z Minecraft et afficher une cible directement sur la carte.

➤

Trouver un lieu selon son besoin

Dans le menu Outils, ouvre Parcours utiles pour afficher rapidement les commerces, institutions, lieux de santé ou lieux de divertissement recensés.

🔗

Partager une position

Depuis l’outil de coordonnées, tu peux copier un lien vers une position précise afin de la partager facilement à un autre joueur.

✎

Faire une remarque

Utilise l’onglet Remarque à droite pour signaler un bug, proposer une idée, corriger une description ou indiquer un lieu manquant et directement générer le message ticket.

✦

Consulter les nouveautés

Le bouton Nouveautés dans la barre du haut permet de suivre les dernières évolutions de l’Atlas et les ajouts prévus.

Atlas Officiel

Bienvenue sur la carte officielle de Karminéa

Cette plateforme a pour but de faciliter l’exploration de Karminéa, aussi bien pour les nouveaux arrivants que pour les habitants déjà installés. Elle permet de retrouver les lieux importants, de consulter les quartiers, les ponts, les rues, les mers et les différents points d’intérêt recensés sur la carte.

N’hésitez pas à parcourir les fonctionnalités disponibles : l’onglet Explorer, les calques, les fiches descriptives, l’aide et les nouveautés.

Si certains points d’intérêt ne sont pas encore recensés, pas de panique : l’Atlas est encore en cours d’enrichissement et de nouveaux lieux seront ajoutés progressivement.

La plateforme est en constante amélioration. Si vous remarquez un bug, un souci d’ergonomie, une limite, une erreur dans un descriptif, une piste d’amélioration, ou bien même tout simplement si vous avez une idée de fonctionnalité, vous pouvez me contacter sur Discord : @Mahyster. Je serai ravi d’échanger avec vous !

Pensez également à consulter régulièrement l’onglet Nouveautés pour suivre les ajouts et les prochaines évolutions de l’Atlas.
