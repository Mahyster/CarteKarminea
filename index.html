Aide rapide

Comment utiliser l’Atlas ?

🖱️

Déplacer la carte

Maintiens le clic gauche sur la carte puis déplace ta souris pour naviguer.

＋

Zoomer et recentrer

Utilise la molette de la souris ou les boutons −, + et Recentrer dans la barre du haut.

☰

Explorer les lieux

Ouvre l’onglet Explorer à gauche pour rechercher un commerce, une institution, un quartier ou un lieu précis.

◎

Afficher les calques

Le panneau en haut à droite permet d’afficher ou masquer les lieux, les ponts, les routes, les océans, les quartiers et les places.

✦

Cliquer sur un élément

Clique sur un marqueur ou sur un élément de carte pour ouvrir sa fiche d’information.

⌖

Mode placement

Active le mode placement pour récupérer facilement des coordonnées en pourcentage sur la carte.

Atlas Officiel

Bienvenue sur la carte officielle de Karminéa

Cette plateforme a pour but de faciliter l’exploration de Karminéa, de retrouver les lieux importants, de consulter les quartiers, les ponts, les rues, les mers et les différents points d’intérêt recensés sur la carte.

N’hésitez pas à parcourir les fonctionnalités disponibles : l’onglet Explorer, les calques, les fiches descriptives, l’aide et les nouveautés.

La plateforme est en constante amélioration. Si vous remarquez un bug, un souci d’ergonomie, une limite, une erreur dans un descriptif,une piste d’amélioration, ou bien même tout simplement si vous avez une idée de fonctionnalité, vous pouvez me contacter sur Discord : @Mahyster. Je serai ravi d'échanger avec vous !

Pensez également à consulter régulièrement l’onglet Nouveautés pour suivre les ajouts et les prochaines évolutions de l’Atlas.
